# План реализации игры Cross Math

## Описание цели
Реализовать игру "Cross Math" с использованием PyQt6. Игра представляет собой кроссворд из математических уравнений, где пропущены некоторые числа. Игроки должны перетаскивать числа из отсортированного списка в правильные пустые ячейки, чтобы решить головоломку.

## Требуется проверка пользователем
> [!IMPORTANT]
> **Уровни сложности**: Я определю уровни сложности следующим образом. Пожалуйста, подтвердите, соответствует ли это вашим ожиданиям:
> - **Легкий**: Маленькая сетка (например, 5x5), простые операции (+, -), уравнения с одной операцией.
> - **Средний**: Средняя сетка (например, 7x7), включает (*), смешанные уравнения с 1-2 операциями.
> - **Сложный**: Большая сетка (например, 9x9), включает (/), в основном уравнения с 2 операциями.
> - **Эксперт**: Большая сетка (например, 11x11), все операции, сложные пересечения.

- Класс `CrossMathGrid`: Управляет состоянием 2D сетки, пересечениями и проверкой.
- Класс `PuzzleGenerator`:
    - Генерирует допустимый макет кроссворда из уравнений.

    - Удаляет числа, чтобы создать "дырки" для заполнения игроком.

## План проверки

### Автоматизированные тесты
- Модульные тесты для генерации `Equation`, чтобы гарантировать математическую корректность.
- Модульные тесты для `CrossMathGrid`, чтобы гарантировать правильное пересечение уравнений.

### Ручная проверка
- Запустить приложение.
- Проверить функциональность Drag and Drop:
    - Перетащить число в пустую ячейку.
    - Перетащить число обратно в список (если разрешено) или поменять числа местами.
- Проверить условие "Победа", когда все ячейки заполнены правильно.
- Проверить, что уровни сложности изменяют размер сетки и сложность.
